# Build Failures

## Common Causes

### Code Errors

Example:

```java
System.out.println("Hello"
```

Compilation fails.

### Dependency Issues

Example:

```xml
Missing Maven Dependency
```

### Test Failures

Unit tests fail.

### Disk Space Issues

Server storage becomes full.

## Troubleshooting Process

Check:

1. Console Output
2. Jenkins Logs
3. Source Code Changes
4. System Resources

## Real-Time Scenario

Build worked yesterday.

Today's build fails.

Recent code commit introduced syntax errors.

## Interview Question

Q: First step when build fails?

Answer:
Check the Console Output.
