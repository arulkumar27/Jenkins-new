# Scenario-Based Questions

## Scenario 1

Build Suddenly Fails

What will you do?

Answer:

1. Check Console Output
2. Check Recent Commits
3. Verify Dependencies
4. Check Agent Status

---

## Scenario 2

Pipeline Stuck in Queue

Answer:

1. Check Executors
2. Check Agent Availability
3. Verify Resource Usage

---

## Scenario 3

GitHub Push Not Triggering Build

Answer:

1. Verify Webhook
2. Check Jenkins Logs
3. Validate Repository Access

---

## Scenario 4

Deployment Failed

Answer:

1. Verify Credentials
2. Check Deployment Logs
3. Verify Network Connectivity
