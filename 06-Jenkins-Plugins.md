# Jenkins Plugins

## What are Plugins?

Plugins extend Jenkins functionality.

Without plugins Jenkins has limited capabilities.

## Popular Plugins

### Git Plugin

Integrates Jenkins with Git repositories.

### Pipeline Plugin

Supports Pipeline as Code.

### Docker Plugin

Integrates Docker builds.

### Kubernetes Plugin

Integrates Kubernetes clusters.

### Blue Ocean

Modern Jenkins Interface.
