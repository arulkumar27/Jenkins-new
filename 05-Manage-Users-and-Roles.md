# Manage Users and Roles

## Why User Management?

Not every user should have administrator access.

## Roles

### Administrator

Permissions:

- Full Access
- Plugin Installation
- Security Configuration

### Developer

Permissions:

- Build Jobs
- View Logs

### Tester

Permissions:

- Trigger Test Jobs
- View Reports

## Security Best Practice

Never share admin credentials.

Use separate accounts.

## Role-Based Access Control

Benefits:

- Security
- Audit Tracking
- Controlled Access

## Real-Time Scenario

Company Team:

10 Developers
3 Testers
2 DevOps Engineers

Each receives different permissions.

## Interview Questions

Q: Why implement RBAC?

Answer:
RBAC restricts access based on user roles and improves security.
