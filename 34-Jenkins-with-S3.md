# Jenkins with S3

## What is S3?

Amazon S3 is an object storage service.

Common Uses:

- Artifact Storage
- Backup Storage
- Log Storage

## Upload Artifact

```bash
aws s3 cp app.jar s3://mybucket/
```

## Download Artifact

```bash
aws s3 cp s3://mybucket/app.jar .
```

## Jenkins Pipeline Example

```groovy
stage('Upload Artifact') {

    steps {
        sh 'aws s3 cp target/app.jar s3://mybucket/'
    }
}
```

## Real-Time Scenario

Build artifacts are stored in S3 before deployment.

## Interview Question

Q: Why integrate S3 with Jenkins?

Answer:
To securely store artifacts, logs, and backups generated by Jenkins.
