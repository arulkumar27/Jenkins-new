# Jenkins Interview Questions

## Basic Questions

### What is Jenkins?

Open-source automation server used for CI/CD.

### What is CI/CD?

CI:
Continuous Integration

CD:
Continuous Delivery/Deployment

### What is a Jenkins Pipeline?

Automated workflow written as code.

### What is Jenkinsfile?

File containing pipeline definitions.

### What are Jenkins Agents?

Machines responsible for executing jobs.

## Intermediate Questions

### Difference Between Freestyle and Pipeline Jobs?

Freestyle:
GUI-Based

Pipeline:
Code-Based

### What is Multi-Branch Pipeline?

Automatically creates jobs for Git branches.

### What are Shared Libraries?

Reusable pipeline code.

## Advanced Questions

### How do you secure Jenkins?

- RBAC
- HTTPS
- Credentials Management
- IAM Roles

### How do you troubleshoot failed builds?

- Console Output
- Logs
- Dependency Verification
