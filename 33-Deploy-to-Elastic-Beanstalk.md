# Deploy to Elastic Beanstalk

## What is Elastic Beanstalk?

AWS Elastic Beanstalk is a Platform as a Service (PaaS) used for application deployment.

AWS manages:

- Infrastructure
- Load Balancing
- Scaling
- Monitoring

## Workflow

Developer Push
↓
Jenkins
↓
Elastic Beanstalk

## Deployment Command

```bash
eb deploy
```

## Benefits

- Easy Deployment
- Auto Scaling
- Managed Environment

## Real-Time Scenario

Small companies use Elastic Beanstalk for application hosting without managing servers.

## Interview Question

Q: Why use Elastic Beanstalk?

Answer:
Elastic Beanstalk simplifies application deployment and infrastructure management.
