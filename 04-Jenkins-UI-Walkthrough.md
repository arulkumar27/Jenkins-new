# Jenkins UI Walkthrough

## Dashboard

Main Jenkins Home Page.

Displays:

- Jobs
- Build History
- Build Queue

## New Item

Used to create:

- Freestyle Jobs
- Pipeline Jobs
- Multi-Branch Pipelines

## Manage Jenkins

Used for:

- Plugin Management
- Security
- Credentials
- Nodes

## Build Queue

Displays jobs waiting for execution.

## Build Executor Status

Shows running builds.

## Real-Time Example

DevOps Engineer checks:

- Failed Builds
- Running Jobs
- Node Status

daily through Jenkins Dashboard.

## Interview Questions

Q: What is Manage Jenkins?

Answer:
Manage Jenkins is the administration area used to configure Jenkins settings and plugins.
