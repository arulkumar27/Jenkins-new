# Security Best Practices

## Principle of Least Privilege

Users should receive only required permissions.

## Secure Jenkins Access

Use:

- HTTPS
- Strong Passwords
- MFA

## Restrict Anonymous Access

Disable public access.

## Keep Plugins Updated

Outdated plugins may contain vulnerabilities.

## Secure Credentials

Store secrets in Jenkins Credentials Store.

## Backup Jenkins

Regular backups reduce recovery time.

## Audit Logging

Track user activities and changes.

## Real-Time Production Checklist

✓ HTTPS Enabled

✓ RBAC Configured

✓ IAM Roles Used

✓ Secrets Protected

✓ Regular Backups

✓ Plugin Updates

## Interview Question

Q: What are important Jenkins security practices?

Answer:

- Least Privilege Access
- HTTPS
- Secure Credentials
- Plugin Updates
- RBAC
- Audit Logging
