# Jenkins Authorization

## What is Authorization?

Authorization determines what a user can do after login.

## Example

Admin:

- Full Control

Developer:

- Build Jobs

Tester:

- View Results

## Role-Based Access Control

Common Roles:

- Admin
- DevOps
- Developer
- Tester

## Benefits

- Controlled Access
- Improved Security
- Audit Compliance

## Real-Time Scenario

Developers cannot modify Jenkins security settings.

Only administrators can.

## Interview Question

Q: Difference between Authentication and Authorization?

Answer:

Authentication:
Who are you?

Authorization:
What can you access?
